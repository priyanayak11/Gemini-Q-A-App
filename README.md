# 🤖 Gemini Q&A App

A simple and interactive **Question & Answer web app** built using **Streamlit**, **LangChain**, and **Google Gemini**.

---

## 🚀 Features

* Ask any question and get instant AI-generated answers
* Built using Google's Gemini model via LangChain
* Clean and simple UI with Streamlit
* Uses environment variables for secure API key handling

---

## 🛠️ Tech Stack

* Python
* Streamlit
* LangChain
* Google Gemini API
* python-dotenv

---

## 📁 Project Structure

project/
│── app.py
│── requirements.txt

---

## 💡 How It Works

1. User enters a question in the UI
2. LangChain formats the input using a prompt template
3. Gemini model generates a response
4. Output parser converts response to plain text
5. Streamlit displays the answer
